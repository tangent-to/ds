## Learning Path

**Beginner** -- Start here if you're new to data science:
1. [Getting Started](getting-started) -- Installation and first analysis
2. [Examples](examples) -- Quick code snippets
3. [Statistics API](api/statistics) -- Statistical tests and GLM

**Intermediate** -- Build on statistical foundations:
1. [Multivariate Analysis API](api/multivariate) -- PCA, LDA, RDA
2. [Machine Learning API](api/machine-learning) -- Models and pipelines

**Advanced** -- Deep dive into the full API:
1. [Core Utilities API](api/core) -- Linear algebra, formulas, optimization
2. [Visualization API](api/visualization) -- Custom plot configurations
